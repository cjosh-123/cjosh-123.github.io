Mask map instruction

First, thank you for using this mask map.Now I'll start explain how to use the buttons in the interface.

Switch to Adult / Child's masks

Each marked point on the map will display the stock status of the pharmacy. The default is to display the stock status of the mask. The button above will switch to display the stock status of the child mask and the button will turn to a green bottom. Allows you to quickly distinguish whether the inventory status of the adult mask is displayed or the inventory status of the child mask.

Move to your location

If you cannot find your current location when looking for a pharmacy, then this button can help you return to your current location immediately. Of course, you must first authorize this map to obtain your current location. Only the current location will be marked when you click for the first time. Location, click back to move the center point of the map to the current location.
